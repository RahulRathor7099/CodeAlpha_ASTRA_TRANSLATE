# CodeAlpha_ASTRA_TRANSLATE


#🌍 Language Translation Tool

A simple and responsive Language Translation web application that allows users to translate text from one language to another using translation APIs.

## 🚀 Features

- Translate text between multiple languages
- User-friendly interface
- Source and target language selection
- Fast translation response
- Responsive web design
- Copy translated text feature (if added)

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Translation API

## 📂 Project Structure

- index.html → Main webpage
- style.css → Styling file
- script.js → Translation logic
